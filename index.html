The MiniSynth Project

A collection of Eurorack-standard synth modules I have built on PCB. Each post will contain links to the schematics as well as the Gerber files for those who would like to have them printed.

MiniOSC: Volt-per-octave CD40106 VCO
An oscillator module based on Moritz Klein's Shapes VCO
MiniVCA: LM13700 VCA
A VCA module based on Juanito Moore's VCA.

Build List

References for modules I have built myself and can confirm that they work. Please bear with me as I am still working on documenting my earlier modules. DM me on IG if you need help. :)

Bridged-T Kick Drum
A kick drum module built around a bridged-t network filter configuration
Phase 90 LFO
Low frequency oscillator based on the MXR Phase 90's LFO block.
8-Step Sequencer
An 8-step sequencer based on the classic Baby-8 design. Works on +9V power.
Active LPF with Resonance
An OpAmp-based low pass filter based Moritz Klein's LPF tutorial. Works on +9V power supply.
AR Envelope Generator
An envelope generator with attack and release controls. Works on +9V power supply.
Arduino-based Master Clock
A clock module based on an Arduino-nano, has multiple divisions and variable pulse widths.
Passive Vactrol VCA
A voltage-controlled attenuator using DIY vactrols
Passive Vactrol LPF
A completely passive low pass filter using DIY vactrols
CD40106 Oscillator Bank
The first analog module I've built using a CD40106 IC. Works on +9V power supply.
