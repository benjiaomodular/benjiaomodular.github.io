A collection of Eurorack-standard synth modules I have designed for PCB. Each post will contain links to the schematics as well as the Gerber files for those who would like to have them printed.

Build List

References for modules I have built myself and can confirm that they work. Please bear with me as I am still working on documenting my earlier modules. DM me on IG if you need help. :)

A circuit to adapt a karaoke effects unit to modular
Wavefolder
A wavefolder module based on Ken Stone's design.
ADSR Envelope Generator
An envelope generator based on Rene Schimtz' "Fastest Envelope in the West"
Arduino-based LFO
A low-frequency oscillator based on HAGIWO's Ardiuno code and schematics.
Overdrive for Modular Synth
I tried to design an overdrive that works on +/-12V power
Arduino-based FM VCO
A freqency-modulation VCO based on HAGIWO's design
PT2399 Delay
An active high pass filter based on a TL072 and a vactrol
Active HPF
An active high pass filter based on a TL072 and a vactrol
Dual Shape LM358 Oscillator
My take on the All About Circuits VCO
NLC Bong0
The Bong0 module by Nonlinearcircuits
Random Triggers and CV
An Arduino-based synth module that produces both randomized CV and randomized gates.
3-Channel Trigger Sequencer
A Baby-8 based sequencer with three trigger outputs for drums
Metallic Hi-Hats
A metallic hi-hat module based on six square wave oscillators.
CD4081 AND Gate Module
An AND logic gate module using a CD4081 CMOS chip.
Resonant Low-Pass Filter
An improvement on my first LPF. This one runs on 18V power.
Bridged-T Snare Drum

Bridged-T Kick Drum
A kick drum module built around a bridged-t network filter configuration
