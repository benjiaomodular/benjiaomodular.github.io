Bill of materials

Capacitors

  • 2.2nF x 1
  • 1uF x 2
  • 100uF x 2
  • 100nF x 3

Diodes

  • 1N4148 x 2
  • 5mm LED x 1

Thermistors

  • 10K NTC Thermistors x 4

Potentiometers

  • B100K x 3
  • 1K trimmer pot x 1

Resistors

  • 100K x 8
  • 1M x 2
  • 1K5 x 1
  • 200K x 1
  • 33K x 1
  • 20K x 1
  • 10K x 3

ICs

  • TL074 x 1
  • CD40106 x 1

Jacks and headers

  • 3.5mm Audio Jack
